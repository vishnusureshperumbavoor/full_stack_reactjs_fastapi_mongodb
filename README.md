# Full Stack ReactJS FastAPI MongoDB app

## Technologies
* ReactJS (frontend)
* FastAPI (backend)
* MongoDB (database)

## Prerequisites
* NodeJS 
* Python
* MongoDB
* VSCode (Code editor)
* Thunder Client (VSCode extension for API testing)


## How to set up
Open client side
```sh
cd client
npm install
npm start
```
Client side will be running at [localhost:3000](https://localhost:3000)  
Open server side
```sh
cd server
pip install -r requirements.txt
uvicorn main:app --reload
```
Server side will be running at [localhost:8000](https://localhost:8000)
